### 项目介绍
技术栈：vue3+pinia+vant

### 项目运行

```shell
git clone https://github.com/Juenfy/cover-wechat-client.git
```

```shell
cd cover-wechat-client
```

```shell
cp .env.example .env
```

```shell
npm install
```
浏览器访问：
```shell
npm run dev
```
桌面端：
```shell
npm run tauri dev
```
桌面端打包：
```shell
npm run tauri build
```
