# <center>仿移动端微信web版</center>

### 项目介绍
采用前端框架vue3+vant，目前只是做了一些页面，还没做功能，会一直持续更新！！！

### 项目运行

```shell
https://github.com/Juenfy/cover-wechat-front.git
```

```shell
cd cover-wechat-front
```

```shell
npm install
```

```shell
npm run dev
```
