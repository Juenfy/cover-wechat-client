# <center>仿移动端微信web版</center>

### 项目介绍
采用前端框架vue3+vant

#### 页面：
1.注册登录

2.聊天列表、聊天窗口、聊天设置

3.发起群聊、扫一扫（https协议）、添加朋友

4.通讯录列表、新的朋友、朋友主页

5.全局通用搜索

等等
#### 暂时支持功能：
1.注册登录

2.朋友搜索申请添加、设置权限、备注等

3.朋友友主页

4.接入websocket

### 项目运行

```shell
git clone https://github.com/Juenfy/cover-wechat-front.git
```

```shell
cd cover-wechat-front
```

```shell
npm install
```

```shell
npm run dev
```
